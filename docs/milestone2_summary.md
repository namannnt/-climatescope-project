# Milestone 2: Data Visualization & Interactive Dashboard

## Overview
This milestone focuses on creating interactive visualizations and a comprehensive Streamlit dashboard to explore the Global Weather Repository dataset. The goal is to make the data insights accessible and interactive for users.

## Objectives
1. Create interactive visualizations using Plotly
2. Build geographic visualizations with Folium maps
3. Develop a comprehensive Streamlit dashboard
4. Enable users to explore weather patterns interactively
5. Visualize air quality trends across countries

## Technologies Used
- **Streamlit 1.29.0**: Interactive web dashboard framework
- **Plotly 5.18.0**: Interactive charts and graphs
