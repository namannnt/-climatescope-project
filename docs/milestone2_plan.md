# Milestone 2: Data Visualization & Interactive Dashboard

## Project: ClimateScope - Global Weather Trends Analysis

---

## Overview
Milestone 2 focuses on creating interactive visualizations and building a comprehensive dashboard to explore global weather patterns, air quality trends, and extreme weather events identified in Milestone 1.

---

## Objectives

### Primary Goals
1. **Interactive Visualizations**: Create dynamic charts using Plotly for weather and air quality analysis
2. **Geographic Mapping**: Build interactive maps using Folium to visualize global weather patterns
3. **Dashboard Development**: Develop a Streamlit-based interactive dashboard for data exploration
4. **Statistical Analysis**: Perform advanced statistical tests and correlation analysis

---

## Planned Deliverables

### 1. Visualization Scripts

#### A. Weather Pattern Visualizations (`scripts/weather_visualizations.py`)
- **Temperature Analysis**
  - Global temperature distribution (histogram)
  - Temperature trends over time (line charts)
  - Temperature by country (box plots)
  - Hottest vs coldest locations (bar charts)

- **Precipitation & Humidity**
  - Precipitation patterns by region
  - Humidity distribution across countries
  - Seasonal precipitation trends

- **Wind & Pressure**
  - Wind speed distribution
  - Wind direction rose charts
  - Atmospheric pressure variations
